Send your Stream records to Slack.

== Description ==

[Stream](https://wordpress.org/plugins/stream/) records activity on your WordPress site in your WordPress database.
This plugin allows you to also send those records to [Slack](https://slack.com/).

You can configure the message with macros.

Based on [Stream to Papertrail](https://wordpress.org/plugins/stream-to-papertrail/)

== Installation ==

1. Search for Stream to Slack in your WordPress dashboard.
2. Install the plugin, and activate it.
3. Set the Incoming Webhook URL, Channel, Emoji Icon and Bot Name for Slack on your Stream settings screen.
